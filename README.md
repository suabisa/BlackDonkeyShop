# BlackDonkeyShop
微信小程序-黑驴商城

**项目介绍：**
基于原生微信小程序框架开发
**已完成功能：**
- 登录授权
- 购物车添加、删除物品功能
- 个人中心展示
- 商品搜索
- 商品收藏
- 查询订单

接口文档：https://www.showdoc.com.cn/128719739414963
